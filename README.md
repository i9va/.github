## Open Source

We share libraries, tools, and internal projects when they can benefit the broader community. We believe in giving back to the ecosystem we build on every day.

<a href="https://github.com/i9va/tracelocal"><img src="https://raw.githubusercontent.com/i9va/tracelocal/main/docs/logo.svg" alt="tracelocal" height="40" /></a>

Zero-config local distributed tracing — straight to your terminal. No Jaeger, no Tempo, no cloud account. Just run one binary.

![Stars](https://img.shields.io/github/stars/i9va/tracelocal?style=flat-square)
![Go](https://img.shields.io/badge/Go-1.24+-00ADD8?style=flat-square&logo=go)
![License](https://img.shields.io/badge/License-Apache%202.0-000000?style=flat-square)
